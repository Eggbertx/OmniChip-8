OmniChip-8
=======
A Chip-8 emulator with heavy emphasis on being as cross-platform as humanly possible

Supported platforms
------ 
### Partial
 * Linux (SDL)
 
### Planned
 * Linux (ncurses)
 * OS X (SDL/ncurses)
 * BSD (SDL/ncurses)
 * Windows/Cygwin (SDL/ncurses)
 * MS-DOS/FreeDOS/DR-DOS
 * Commodore 64
 * Apple ][
